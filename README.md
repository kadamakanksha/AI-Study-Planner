 # 📚 Smart AI Study Planner

## 🚀 Overview
This project is an AI-powered study planner that generates personalized study schedules based on user goals.

## 🎯 Vertical
Education / Productivity Assistant

## 🧠 Approach
- Takes user input goal
- Uses weighted logic to prioritize subjects
- Generates a dynamic study plan
- Simulates intelligent decision-making

## ⚙️ Features
- Dynamic plan generation
- Difficulty-based scheduling
- Simple UI
- Scalable architecture

## ☁️ Google Services
- Designed to integrate with Google Calendar API for scheduling tasks

## 📌 Assumptions
- Default duration: 30 days
- Subjects predefined

## ▶️ How to Run
1. Open index.html
2. Enter goal
3. Click generate
